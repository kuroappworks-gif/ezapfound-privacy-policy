ezapfound 개인정보 처리방침

본 개인정보 처리방침은 KURO APP WORKS에서 개발한 모바일 기기용 소프트웨어 애플리케이션인 "ezapfound"(이하 "본 앱")의 이용에 관한 것입니다. 본 앱은 설치된 앱을 정리하고 중요한 연락처에 대한 빠른 접근 단축어(Shortcuts)를 생성하기 위해 설계된 심플한 홈 런처입니다.

1. 사용자가 제공하는 정보

본 앱은 개인정보를 당사의 서버로 수집, 전송 또는 저장하지 않습니다. 다만, 본 앱에서는 사용자가 수동으로 다음과 같은 정보를 입력할 수 있으며, 이는 사용자의 기기 내에만 로컬로 저장됩니다.

  • 연락처 단축어 정보: SOS/연락처 단축어를 생성할 목적으로 사용자가 제공한 이름과 전화번호. 이 데이터는 '원탭 다이얼' 기능을 활성화하기 위한 용도로만 사용됩니다.
  • 리마인더/캘린더 정보: 사용자가 생성한 일정의 제목, 상세 내용, 날짜, 시간, 설정 색상 등의 정보. 이는 리마인더 알림 및 캘린더 표시 기능만을 위해 사용됩니다.

2. 자동으로 수집되는 정보 / 비개인적 데이터

본 앱은 런처의 주요 기능을 제공하기 위한 목적으로만 다음과 같은 데이터를 사용자의 기기 내에 로컬로 수집하고 저장합니다.

  • 설치된 앱 정보: 기기에 설치된 앱의 패키지명, 앱 라벨(표시 이름), 설치 일시.
  • 사용자 설정 및 환경 설정: 카테고리 이름, 앱 정리/정렬 순서, 테마/글꼴 크기 설정, 배경 이미지 URI 등의 구성 데이터.
  • 백업 데이터: 백업 기능을 사용할 경우, 사용자의 지시에 따라 카테고리와 앱 배치 정보를 포함한 JSON 파일이 기기의 저장 공간에 생성됩니다.

상기 로컬 저장 데이터는 아래 제4항에 기재된 제3자 서비스에 의해 요구되는 경우를 제외하고, 당사의 서버나 제3자에게 전송되지 않습니다.

3. 사용되는 권한 및 목적

본 앱은 다음과 같은 권한을 필요로 하며, 명시된 목적을 위해서만 사용합니다.

  • android.permission.INTERNET: 광고 표시 및 동의 관리를 위해 필요합니다.
  • com.google.android.gms.permission.AD_ID: Google Mobile Ads SDK(AdMob)가 광고 및 분석 목적으로 사용하기 위해 필요합니다.
  • android.permission.RECORD_AUDIO: 사용자가 음성 검색/마이크 아이콘을 자발적으로 탭한 경우에만 사용됩니다. 음성 데이터는 기기 표준 음성 인식 서비스에 의해 처리되며, 본 앱에 의해 녹음되거나 저장되지 않습니다.
  • android.permission.SCHEDULE_EXACT_ALARM: 사용자가 설정한 리마인더를 지정된 정확한 시간에 알리기 위해 사용됩니다.
  • android.permission.USE_FULL_SCREEN_INTENT: 중요한 리마인더(투약 시간 등)를 기기가 잠겨 있는 상태에서도 확실하게 전체 화면으로 알리기 위해 사용됩니다.
  • android.permission.POST_NOTIFICATIONS: 앱의 알림(리마인더 및 알람)을 표시하기 위해 사용됩니다.

4. 제3자 서비스

본 앱은 다음과 같은 제3자 서비스 제공업체를 사용합니다.

  • Google AdMob: 광고 송출을 위해 사용합니다.
  • Google User Messaging Platform (UMP) SDK: 개인화된 광고에 대한 사용자의 동의를 관리하기 위해 사용합니다(GDPR/CCPA 준수).

이러한 서비스는 개인화된 광고 송출, 노출 빈도 제한(Frequency Capping) 및 광고 보고 목적으로 정보를 수집할 수 있습니다. 자세한 내용은 Google의 개인정보 처리방침을 참조하시기 바랍니다.

5. 데이터 보관 정책

로컬에서 생성된 모든 데이터(앱 목록, 카테고리, 리마인더, 설정)는 본 앱이 삭제되거나 사용자가 수동으로 데이터를 삭제할 때까지 사용자의 기기에 저장됩니다. 사용자가 생성한 백업 파일은 앱 삭제 후에도 기기의 저장 공간에 남아 있을 수 있으며 수동으로 삭제해야 합니다.

6. 아동의 개인정보 보호

본 앱은 일반인을 대상으로 설계되었으며, 13세 미만의 아동으로부터 의도적으로 데이터를 수집하거나 아동을 대상으로 마케팅을 하지 않습니다.
