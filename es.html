Política de privacidad de ezapfound

Esta política de privacidad se aplica a la aplicación de software «ezapfound» (en adelante, la «Aplicación») para dispositivos móviles, creada por KURO APP WORKS. La Aplicación es un lanzador de inicio sencillo diseñado para organizar aplicaciones instaladas y crear atajos de acceso rápido para contactos importantes.

1. Información proporcionada por el usuario

La Aplicación no recopila, transmite ni almacena ninguna información personal en nuestros servidores. Sin embargo, los usuarios pueden introducir manualmente la siguiente información, que se almacena únicamente de forma local en su dispositivo:

  • Información de atajos de contacto: Nombres y números de teléfono proporcionados por el usuario con el fin de crear atajos de SOS/contacto. Estos datos se utilizan exclusivamente para habilitar la función de «Marcación con un toque».
  • Información de recordatorios/calendario: Títulos, detalles, fechas, horas y ajustes de color para las tareas creadas por el usuario. Estos datos se utilizan únicamente para las funciones de notificación de recordatorios y visualización del calendario.

2. Información recopilada automáticamente / Datos no personales

La Aplicación recopila y almacena los siguientes datos localmente en su dispositivo, únicamente con el propósito de proporcionar las funciones principales del lanzador:

  • Información de aplicaciones instaladas: Nombres de paquetes, etiquetas de aplicación (nombres para mostrar) y fechas/horas de instalación de las aplicaciones en el dispositivo.
  • Ajustes y preferencias del usuario: Datos de configuración como nombres de categorías, organización/orden de apps, ajustes de tema/tamaño de fuente y URIs de imágenes de fondo.
  • Datos de copia de seguridad: Al usar la función de copia de seguridad, se crea un archivo JSON en el almacenamiento del dispositivo con la información de categorías y ubicación de apps, basado en las instrucciones del usuario.

Los datos almacenados localmente mencionados anteriormente nunca se envían a nuestros servidores ni a terceros, excepto cuando lo requieran los servicios de terceros descritos en la Sección 4.

3. Permisos utilizados y sus finalidades

La Aplicación requiere los siguientes permisos y los utiliza únicamente para los fines indicados:

  • android.permission.INTERNET: Necesario para mostrar anuncios y gestionar el consentimiento.
  • com.google.android.gms.permission.AD_ID: Necesario para el SDK de anuncios de Google (AdMob) con fines publicitarios y de análisis.
  • android.permission.RECORD_AUDIO: Se utiliza solo cuando el usuario toca voluntariamente el icono de búsqueda por voz/micrófono. Los datos de audio son procesados por el servicio de reconocimiento de voz estándar del dispositivo y la Aplicación no los graba ni almacena.
  • android.permission.SCHEDULE_EXACT_ALARM: Se utiliza para activar los recordatorios configurados por el usuario en la hora exacta especificada.
  • android.permission.USE_FULL_SCREEN_INTENT: Se utiliza para mostrar alertas a pantalla completa para recordatorios importantes (ej: toma de medicación), garantizando que se comuniquen incluso con el dispositivo bloqueado.
  • android.permission.POST_NOTIFICATIONS: Se utiliza para mostrar notificaciones de la aplicación (recordatorios y alarmas).

4. Servicios de terceros

La Aplicación utiliza los siguientes proveedores de servicios de terceros:

  • Google AdMob: Para la entrega de anuncios.
  • Google User Messaging Platform (UMP) SDK: Para gestionar el consentimiento del usuario para publicidad personalizada (cumple con GDPR/CCPA).

Estos servicios pueden recopilar información para fines como la entrega de anuncios personalizados, limitación de frecuencia e informes publicitarios. Para más detalles, consulte la política de privacidad de Google.

5. Política de retención de datos

Todos los datos generados localmente (listas de apps, categorías, recordatorios, ajustes) permanecen en el dispositivo del usuario hasta que la Aplicación se desinstala o el usuario elimina los datos manualmente. Los archivos de copia de seguridad pueden permanecer en el almacenamiento tras la desinstalación y deben eliminarse manualmente.

6. Privacidad infantil

La Aplicación está diseñada para un público general y no recopila datos de niños menores de 13 años a sabiendas, ni se dirige a ellos con fines de marketing.

7. Cambios en esta política de privacidad

Podemos actualizar nuestra política de privacidad periódicamente. Le notificaremos cualquier cambio publicando la nueva política de privacidad en esta página.
